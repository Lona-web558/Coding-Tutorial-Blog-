JavaScript Variables

Learn what variables are in Javascript, and how to create them.

What is are Variables?

A variable is a data storing object. Variables have strict rules on how they are to be written.

Creating a Variable

Here's how you can create a variable in JavaScript:


let fruits;
var fruits;
let x;
var y;

Assigning a value to a variable

You can assign values to variables. As shown below. 


                        	let fruits;
                        //assign value to variable
                        fruits = apples, bananas, strawberries;

Try it Yourself

Use the console below to practice working with variables:

let x = 10; let y = 10; let ans = x + y; console.log(ans); // output 20
